# Navoiy Uy Sotuv Sayti

Mobilga mos bir sahifalik uy sotuv sayti.

## Telegram Lead Sozlash

1. `config.sample.php` faylidan nusxa olib `config.php` deb nomlang.
2. `bot_token` joyiga BotFather bergan tokenni yozing.
3. `chat_id` joyiga lead keladigan Telegram chat ID ni yozing.
4. Saytni PHP ishlaydigan hostingga yuklang.

Forma `send-lead.php` orqali Telegram botga yuboradi. `config.php` maxfiy fayl bo'lgani uchun `.gitignore`ga qo'shilgan.
